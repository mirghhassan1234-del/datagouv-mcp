## 📋 About datagouv-mcp

datagouv-mcp is the official server for the Model Context Protocol (MCP). It helps AI chatbots and users search, explore, and analyze datasets from data.gouv.fr. This is the French government’s national open data platform. 

With datagouv-mcp, you can talk to AI chatbots and get answers using real public data. This allows you to understand large datasets without needing technical skills.
